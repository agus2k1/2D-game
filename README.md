# 2D-game
The objective of the game is to make the highest score you can. You have 3 lives and 1 superpower

Controls
Movement = Arrowkeys /
Superpower = Ctrl /
See Hitbox = "D"

If you destroy an enemy you get +1 points, if you get hit by and enemy you lose -4 points and if you sit (freezing the game) you lose -3 points.
